# Tic Tac Toe

A React + Vite tic tac toe game ready for static hosting.

## Run locally

```bash
npm install
npm run dev
```

## Build for deploy

```bash
npm run build
```

The production files are created in `dist`.

## Deploy online

Use these settings on Vercel, Netlify, Render static site, or similar hosts:

- Build command: `npm run build`
- Publish/output directory: `dist`
- Install command: `npm install`

For GitHub Pages, deploy the contents generated by `npm run build` from the `dist` folder.
